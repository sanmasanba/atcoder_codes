# ベンチマーク

## ルール

- 比較は必ず同一条件で行う。
- ベースラインと候補の 2 点比較を基本とする。
- 指標は平均・最良・最悪・実行時間を残す。

## 記録フォーマット

- 実行条件
- ベースライン
- 候補
- 差分
- 採用判断
