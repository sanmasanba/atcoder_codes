# 新・典型問題集

2021年以降の AtCoder Beginner Contest を題材に、近年のABCで必要になる考え方を身につけるための問題集です。

## 対象者

- 典型90題を一通り終えた、または並行して取り組んでいる人
- ABC の C〜F 問題を安定して解けるようになりたい人
- 解法を知っているだけでなく、問題文から使う考え方を選ぶ練習をしたい人

## 使い方

1. [問題一覧](problems/index.md)から、今の自分に近いレベルを1問選びます。
2. AtCoder の問題を読み、まず自力で考えます。
3. 解けなかった場合だけ、このサイトの解説を開きます。
4. 解説末尾の関連問題で、同じ考え方を使えるか確かめます。

問題一覧には典型名・カテゴリ・解法の手がかりを載せていません。先に答えを連想しない状態で問題を読むことを大切にしています。

## 典型90題との関係

典型90題を置き換えるものではありません。基礎的な定石を学ぶ典型90題の次に、近年のABCで定石がどのように組み合わされて出題されるかを学ぶための続編です。

## 更新方針

問題は固定しません。新しいABCから学習効果の高い問題を追加し、より良い代表問題が見つかった場合は差し替えます。変更内容は[更新履歴](changelog.md)に記録します。

!!! note "難度について"
    レベルはAtCoder Problemsなどの推定難度を参考にした目安です。コンテスト中の問題番号や典型カテゴリではありません。
