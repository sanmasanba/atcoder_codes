# 063

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc214/tasks/abc214_e)

## 何に着目するか

各区間へ異なる整数を割り当てるには、締切が早い区間から埋めるのが安全です。

## 解法方針

左端順に区間を追加し、現在時刻で使える区間を右端の最小ヒープに入れます。最も早く閉じる区間へ現在時刻を割り当てます。

## 実装要点

ヒープが空なら次の区間の左端まで時刻を飛ばします。右端より時刻が進んだら不可能です。

## 計算量

`O(N log N)` です。

## 典型・関連問題

典型は **締切最優先の区間スケジューリング** です。関連: [ABC325 D](https://atcoder.jp/contests/abc325/tasks/abc325_d)
