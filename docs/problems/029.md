# 029

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc233/tasks/abc233_f)

## 何に着目するか

辺上の交換で置換を直す問題です。連結成分をまたぐ値は移動できません。

## 解法方針

最初にUnion-Findで必要な値と頂点が同じ連結成分にあるか確認します。各成分の全域木を作り、葉側から目的の値を根方向へ運ぶことで交換列を構成します。

## 実装要点

交換した辺番号を順に記録します。木の親子関係と、各値が今いる頂点を常に更新します。

## 計算量

`O(N + M)` に近い時間で構成できます。

## 典型・関連問題

典型は **全域木上の置換の構成** です。関連: [ABC251 F](https://atcoder.jp/contests/abc251/tasks/abc251_f)
