# 076

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc251/tasks/abc251_e)

## 何に着目するか

円環上の選択は、一つ目を選ぶ・選ばないの二通りへ固定すると直線DPになります。

## 解法方針

最初の要素を選ぶ場合と選ばない場合に分け、各位置で自分を選ぶか直前に任せるかの最小コストをDPします。

## 実装要点

最後と最初の制約が各ケースで満たされるよう初期値を変えます。

## 計算量

`O(N)` です。

## 典型・関連問題

典型は **環状配列を端点固定で解くDP** です。関連: [ABC210 D](https://atcoder.jp/contests/abc210/tasks/abc210_d)
