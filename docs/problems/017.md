# 017

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc257/tasks/abc257_d)

## 何に着目するか

ある能力値で全地点へ到達できるかは、その値を大きくしても壊れない性質です。

## 解法方針

能力値を固定し、到達可能な有向辺を作って各始点から探索します。全頂点に届く始点が存在するかを判定し、この真偽に対して二分探索します。

## 実装要点

判定には距離の二乗と `p[i] * s` を比較して浮動小数点を避けます。上界は十分大きい整数を取ります。

## 計算量

判定は `O(N^3)`、全体は `O(N^3 log C)` です。

## 典型・関連問題

典型は **答えで二分探索＋到達判定** です。関連: [ABC236 E](https://atcoder.jp/contests/abc236/tasks/abc236_e)
