# 074

## 問題リンク

[ABC247 E - Max Min](https://atcoder.jp/contests/abc247/tasks/abc247_e)

## キーワード

部分配列の条件は、右端ごとに最後の境界位置から数える

## 何に着目するか

条件は「最大値が `X`、最小値が `Y`」です。右端を固定すると、左端がどこまでなら範囲外の値を含まず、どこまでなら `X` と `Y` の両方を含むかが、最後に現れた位置だけで決まります。

## 解法方針

左から右へ右端 `r` を動かし、次の三つを記録します。

|変数|意味|
|---|---|
|`bad`|`A[i]<Y` または `A[i]>X` の最後の位置|
|`lastX`|`A[i]==X` の最後の位置|
|`lastY`|`A[i]==Y` の最後の位置|

右端 `r` で有効な左端 `l` は、範囲外を避けるため `l>bad`、`X,Y` を両方含むため `l≤min(lastX,lastY)` を満たします。よって個数は

```text
max(0, min(lastX,lastY) - bad)
```

```text
bad | ... 有効な l ... | min(lastX,lastY) | ... r
      ↑ここからここまでが選べる
```

## tips

### 実装

三変数を全て `-1` で初期化します。`A[r]` を読んで三変数を更新してから、その時点の寄与を答えへ加えます。`X==Y` の場合も式はそのまま動きます。

### よくある誤り

- `lastX`, `lastY` の大きい方を使う。両方含むには早いほうまで左端を戻す必要があります。
- `bad` を含めて左端にする。範囲外の値が入るので `bad+1` 以降です。
- `lastX` / `lastY` が未出現のとき負の寄与を加える。`max(0,...)` を取ります。

### 計算量

配列を一度走査するだけなので、時間 `O(N)`、追加メモリ `O(1)` です。
