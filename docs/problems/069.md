# 069

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc232/tasks/abc232_e)

## 何に着目するか

二点の正確な座標ではなく、目標と行・列が一致するかだけで遷移が決まります。

## 解法方針

目標と同じ・行のみ同じ・列のみ同じ・どちらも異なる、の4状態でDPします。各状態からの遷移数を数えます。

## 実装要点

初期状態は始点と目標の関係で一つに決まります。法の下で更新します。

## 計算量

`O(K)` です。

## 典型・関連問題

典型は **対称性による状態圧縮DP** です。関連: [ABC310 E](048.md)
