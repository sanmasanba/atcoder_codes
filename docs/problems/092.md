# 092

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc303/tasks/abc303_e)

## 何に着目するか

指定された木構造では、葉から一定距離の頂点の次数だけを集めればよいです。

## 解法方針

葉を始点にBFSし、深さが条件に合う頂点の次数を集めます。訪問済みにして重複を防ぎます。

## 実装要点

木であることを利用し、葉の一つからの距離の剰余で分類します。

## 計算量

`O(N)` です。

## 典型・関連問題

典型は **葉を基準にした木の距離層分類** です。関連: [ABC240 E](030.md)
