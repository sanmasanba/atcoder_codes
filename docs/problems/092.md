# 092

## 問題リンク

[ABC303 E - A graph problem](https://atcoder.jp/contests/abc303/tasks/abc303_e)

## キーワード

規則的な木分解は、葉からの距離を法で分類して中心を見つける

## 何に着目するか

与えられる木は、スターの中心どうしを長さ 3 の経路でつないだ形です。ある葉から見ると、スター中心は必ず距離が `1 mod 3` の層に並びます。したがって一つの葉を根にして BFS し、その層の次数を集めればよいです。

## 解法方針

任意の葉を一つ選んで BFS し、各頂点の葉からの距離 `dist` を求めます。`dist % 3 == 1` の頂点がスターの中心なので、その次数を答えへ加えます。

```text
leaf -- center -- leaf
  0       1       2

次の中心は距離 4, 7, ...
→ 中心は常に 1 mod 3
```

中心の次数は対応するスターの大きさです。集めた次数を昇順に並べて出力します。

## tips

### 実装

次数 1 の頂点を見つけ、そこから通常の BFS/DFS をします。木なので親へ戻らないよう訪問済みを持てば全頂点を一度ずつ探索できます。

### よくある誤り

- 全ての葉から BFS する。一つの葉で距離の剰余が定まります。
- `dist%3==0` や 2 の頂点を中心として集める。葉を距離 0 としたとき中心は 1 です。
- 中心の個数を出す。出力するのは各中心の次数です。

### 計算量

BFS と次数集計を一回ずつ行うので、時間・メモリともに `O(N)` です。
