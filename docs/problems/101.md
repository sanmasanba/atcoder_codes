# 101

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc319/tasks/abc319_e)

## 何に着目するか

各バスの周期が小さいため、全経路の待ち時間は出発時刻の剰余だけで決まります。

## 解法方針

周期の最小公倍数をPとして、各 `0..P-1` の出発時刻に対する到着までの追加時間を前計算します。

## 実装要点

クエリ時は出発時刻をPで割った余りを使い、前後の徒歩時間を加えます。

## 計算量

前処理 `O(PN)`、各クエリ `O(1)` です。

## 典型・関連問題

典型は **小さい最小公倍数による周期状態前計算** です。関連: [ABC300 F](043.md)
