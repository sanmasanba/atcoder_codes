# 044

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc302/tasks/abc302_e)

## 何に着目するか

辺の追加・削除に対し、次数0の頂点数だけを知りたいクエリです。

## 解法方針

各頂点の隣接集合と次数を持ちます。辺を追加・削除する直前後に、端点が孤立しているかを見て全体カウンタを更新します。

## 実装要点

削除する辺は必ず存在する前提を使います。両端点を個別に更新します。

## 計算量

各操作は期待 `O(1)` です。

## 典型・関連問題

典型は **局所的な次数変化で全体数を保つ** 手法です。関連: [ABC378 E](https://atcoder.jp/contests/abc378/tasks/abc378_e)
