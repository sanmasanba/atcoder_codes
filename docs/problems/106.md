# 106

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc324/tasks/abc324_e)

## 何に着目するか

二文字列を連結して目標列を部分列にする条件は、前から一致できる長さと後ろから一致できる長さで判定できます。

## 解法方針

各文字列について目標のprefix一致長とsuffix一致長を求めます。prefix長をソートし、各suffixに必要なprefix数を二分探索します。

## 実装要点

後ろ側は文字列・目標列を反転して同じ処理にできます。

## 計算量

`O(total length + N log N)` です。

## 典型・関連問題

典型は **前後の部分列一致長の合成** です。関連: [ABC287 E](035.md)
