# 046

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc308/tasks/abc308_f)

## 何に着目するか

品物とクーポンを高い順に処理すると、今使えるクーポンのうち最良のものだけが必要です。

## 解法方針

品物を価格降順、クーポンを必要価格降順にソートします。各品物で使える割引額を最大ヒープへ入れ、最大を一枚だけ使います。

## 実装要点

クーポンは使用済みにしたらヒープから取り出します。符号反転でPythonの最小ヒープを最大ヒープとして使えます。

## 計算量

`O((N + M) log M)` です。

## 典型・関連問題

典型は **ソート走査と利用可能候補の最大化** です。関連: [ABC245 E](006.md)
