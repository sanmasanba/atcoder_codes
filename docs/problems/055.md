# 055

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc278/tasks/abc278_e)

## 何に着目するか

切り抜き外に残る色は、全体出現範囲が切り抜きに収まらない色です。

## 解法方針

各色の出現行・列の最小最大を前処理します。各切り抜き位置で、その範囲を完全に含む色数を二次元imos法で数えます。

## 実装要点

色ごとに「その色が消える切り抜き左上」の長方形へ加算します。

## 計算量

`O(HW + N)` 程度です。

## 典型・関連問題

典型は **出現範囲から作る二次元imos** です。関連: [ABC005 D](https://atcoder.jp/contests/abc005/tasks/abc005_d)
