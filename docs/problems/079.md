# 079

## 何に着目するか

回転量に対する各要素のコストは折れ線で、傾きが変わる点だけ更新すればよいです。

## 解法方針

各要素のコスト変化を区間加算として差分配列へ入れます。傾きの差分を二回累積して全回転量のコストを復元します。

## 実装要点

環状区間を通常区間へ分割します。変化点の位置を剰余で正しく扱います。

## 計算量

`O(N)` です。

## 典型・関連問題

典型は **巡回パラメータへの傾きimos** です。関連: [ABC338 E](https://atcoder.jp/contests/abc338/tasks/abc338_e)
