# 117

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc335/tasks/abc335_e)

## 何に着目するか

同じ値の頂点は自由に行き来できるため、先に縮約すると値が増加するDAGになります。

## 解法方針

同値辺でUnion-Find併合し、成分を値順に処理します。小さい値から入る辺だけでDP値を更新します。

## 実装要点

始点成分のDPを1にします。終点が未到達なら0です。

## 計算量

`O((N+M)α(N) + N log N)` です。

## 典型・関連問題

典型は **同値縮約＋値順DAG DP** です。関連: [ABC224 E](066.md)
