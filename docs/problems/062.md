# 062

## 問題リンク

[ABC207 D - Congruence Point Sets](https://atcoder.jp/contests/abc207/tasks/abc207_d)

## キーワード

回転・合同判定は、基準ベクトルとの内積・外積で比較する

## 何に着目するか

平行移動は、両方の点集合で一つずつ基準点を原点へ移せば消せます。さらに、基準点から別の一点へのベクトルの対応を決めれば、回転も一意に決まります。

浮動小数点で回転行列を作ると誤差が出るため、内積・外積を使って整数のまま「回転後に一致するか」を判定します。

## 解法方針

`S` の基準点 `s0`、`T` の基準点 `t0` を選びます。`S` の別点 `s1` と `T` の別点 `t1` を対応させ、

```text
u = s1 - s0 = (ux, uy)
v = t1 - t0 = (vx, vy)
```

とします。まず `|u|²==|v|²` でなければこの対応では不可能です。

任意の `s` について `w=s-s0` とすると、`w` が回転後に `z=t-t0` になる条件は、回転で保存される内積・外積から

```text
dot(w,u)   = dot(z,v)
cross(w,u) = cross(z,v)
```

で判定できます。`T` 側の各相対ベクトルについて `(dot(z,v), cross(z,v))` を集合に入れておき、`S` 側の各ベクトルで同じ組があるかを調べます。

```text
s0 を原点に移す     t0 を原点に移す
      u  ── 回転 ──>  v
他の全ベクトルも同じ回転で対応するか確認
```

`N=1` なら任意の一点集合同士は一致可能です。

## tips

### 実装

基準 `s0` を固定し、`t0`、`s1`、`t1` の候補を全探索します。座標差・内積・外積は 64 bit 整数で扱います。`T` の相対ベクトル集合を `set` にして、各候補対応で全点を確認します。

同じ点は存在しないため、`s1≠s0`, `t1≠t0` を選べます。

### よくある誤り

- 回転後の座標を `cos`, `sin` で計算する。丸め誤差で集合検索が壊れます。
- 距離だけを比較する。鏡映（裏返し）まで許してしまう可能性があります。内積と符号付き外積で向きを保ちます。
- 平行移動を消さずに絶対座標を比較する。

### 計算量

基準点・対応ベクトルの候補と全点確認を合わせて `O(N^3)`、メモリは `O(N)` です。
