# 062

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc207/tasks/abc207_d)

## 何に着目するか

平行移動・回転後の一致は、基準点からの相対ベクトルで判定できます。

## 解法方針

一つの点を原点へ移し、別の一点を対応させて回転を決めます。全点の相対位置が一致するかをsetで確認します。

## 実装要点

浮動小数点を避け、内積・外積による回転後ベクトルの整数比較を使います。

## 計算量

`O(N^3)` です。

## 典型・関連問題

典型は **基準ベクトルで固定する合同判定** です。関連: [ABC218 C](https://atcoder.jp/contests/abc218/tasks/abc218_c)
