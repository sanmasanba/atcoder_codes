# 065

## 問題リンク

[ABC221 F - Diameter set](https://atcoder.jp/contests/abc221/tasks/abc221_f)

## キーワード

木の最遠頂点対は、直径中心から等距離の枝の組合せで数える

## 何に着目するか

距離が最大になる二頂点は木の直径の両端です。直径の中心を基準にすると、両端は中心から同じ距離にあり、異なる枝に属します。中心が頂点か辺かで場合分けします。

## 解法方針

二回の DFS/BFS で直径を求め、親を使って直径パスを復元します。直径長を `L` とします。

|L の偶奇|中心|数えるもの|
|---|---|---|
|偶数|一つの中心頂点 c|各隣接枝で距離 `L/2` の頂点数|
|奇数|中心辺 `(u,v)`|u 側・v 側で距離 `(L-1)/2` の頂点数の積|

偶数の場合、各枝の該当頂点数を `cnt1,cnt2,...` とすると、異なる枝から二点を選ぶ数は

```text
(Σcnt)^2 - Σcnt^2
-----------------
        2
```

です。同じ枝の二点では、中心を通らず距離が直径になりません。

```text
       cnt1
        \ 
         c -- cnt2
        /
      cnt3
異なる枝を一つずつ選ぶ
```

## tips

### 実装

中心頂点から各隣へ DFS し、中心を通らず指定深さにある頂点を数えます。奇数の場合は中心辺を一時的に切った二側で深さを数えます。

答えは二頂点対数なので 64 bit 整数を使います。

### よくある誤り

- 同じ枝から二点を選ぶ組も加える。距離は直径未満です。
- 直径長の偶奇で中心が辺になるケースを忘れる。
- 直径そのものの頂点数と辺数を混同する。距離は辺数です。

### 計算量

直径探索・各枝の深さ探索を合わせて、時間・メモリともに `O(N)` です。
