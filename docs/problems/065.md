# 065

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc221/tasks/abc221_f)

## 何に着目するか

最長距離の頂点対は木の直径の両端であり、中心からの分岐で数え上げられます。

## 解法方針

二回のDFSで直径を求めます。直径長の偶奇に応じて中心を一つまたは二つに分け、中心から所定距離の頂点数を掛け合わせます。

## 実装要点

同じ枝から二頂点を選ぶ場合を除外します。直径の復元用に親も保存します。

## 計算量

`O(N)` です。

## 典型・関連問題

典型は **木の直径中心による距離条件の数え上げ** です。関連: [ABC267 F](https://atcoder.jp/contests/abc267/tasks/abc267_f)
