# 053

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc276/tasks/abc276_f)

## 何に着目するか

二つの独立な一様選択の期待値は、過去要素との比較結果を累積すれば更新できます。

## 解法方針

値の個数と値和をFenwick Treeで持ち、新要素と既存要素の `max` の寄与を加算します。分母の逆元を掛けます。

## 実装要点

値を座標圧縮し、等しい値を含む範囲を正しく数えます。

## 計算量

`O(N log N)` です。

## 典型・関連問題

典型は **期待値の逐次更新とFenwick Tree** です。関連: [ABC221 E](https://atcoder.jp/contests/abc221/tasks/abc221_e)
