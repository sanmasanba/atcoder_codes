# 039

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc280/tasks/abc280_f)

## 何に着目するか

辺が二頂点間の値の差を表すグラフでは、同一成分内の相対値を持って併合できます。

## 解法方針

重み付きUnion-Findで各頂点から根までの差を管理します。矛盾する閉路を含む成分は別途印を付け、クエリ時に判定します。

## 実装要点

併合時の符号を式で確認します。矛盾を検出しても成分自体は併合済みとして扱います。

## 計算量

各操作はならし `O(α(N))` です。

## 典型・関連問題

典型は **重み付きUnion-Findと矛盾検出** です。関連: [ABC328 F](https://atcoder.jp/contests/abc328/tasks/abc328_f)
