# 047

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc309/tasks/abc309_e)

## 何に着目するか

祖先から与えられる効果は、子へ一段進むごとに残り回数が減ります。

## 解法方針

各頂点に直接指定された残り回数の最大値を置き、根からDFSします。親から来た残り回数と自分の値の最大を子へ一つ減らして渡します。

## 実装要点

残り回数が0でも現在頂点は対象です。負の値は効果なしとして扱います。

## 計算量

`O(N + M)` です。

## 典型・関連問題

典型は **木の根から伝播するmax DP** です。関連: [ABC138 D](https://atcoder.jp/contests/abc138/tasks/abc138_d)
