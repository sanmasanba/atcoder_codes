# 114

## 問題リンク

[ABC332 E - Lucky bag](https://atcoder.jp/contests/abc332/tasks/abc332_e)

## キーワード

分散の定数項を除き、グループ和の二乗和を最小化する部分集合分割 DP

## 何に着目するか

各グループの重さ和を `s_i`、全体平均を `avg` とすると、分散は

```text
(1/D) Σ (s_i-avg)^2 = (1/D)Σs_i^2 - avg^2
```

`avg^2` は分け方によらない定数です。したがって各グループ和の二乗和だけを最小化すればよいです。

## 解法方針

部分集合 `mask` の重さ和 `sum[mask]` を前計算し、コスト `cost[mask]=sum[mask]^2` を作ります。`dp[d][mask]` を「mask の荷物を d グループへ分けた二乗和最小値」とします。

```text
dp[d+1][mask | sub]
  = min(dp[d+1][...], dp[d][mask] + cost[sub])
```

`sub` はまだ使っていない荷物の部分集合です。空グループも許されるため `sub=0` を選んでもよく、最終的に全荷物を含む mask の値から `avg^2` を引きます。

```text
荷物集合 → D 個の部分集合へ分割
目的: Σ(各部分集合の和)^2 を最小化
```

## tips

### 実装

部分集合和は `sum[mask]=sum[mask without lsb]+W[lsb]` で `O(2^N)` 前計算できます。各 d,mask で残り集合の部分集合を列挙します。

浮動小数点で DP してもよいですが、二乗和を整数で持ち最後に実数へ変換すると安定します。

### よくある誤り

- 平均をグループごとに変える。平均は全体重さ/D で固定です。
- 空グループを禁止する。問題では許されます。
- 二乗和だけをそのまま出力する。最後に `1/D` と `-avg^2` を反映します。

### 計算量

部分集合列挙により `O(D3^N)`、メモリ `O(D2^N)` です。
