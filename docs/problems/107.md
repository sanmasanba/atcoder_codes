# 107

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc325/tasks/abc325_e)

## 何に着目するか

移動手段の切替は一度でよく、切替点を全探索する代わりに両側最短路を前計算できます。

## 解法方針

車だけの始点距離と、列車だけの終点までの距離をそれぞれダイクストラ法で計算し、各切替点で和を最小化します。

## 実装要点

終点からの距離は、列車コストで逆向きに考えるか同じ完全グラフで始点を終点に替えて計算します。

## 計算量

`O(N^2 log N)` です。

## 典型・関連問題

典型は **切替点を介する二本の最短路合成** です。関連: [ABC252 E](013.md)
