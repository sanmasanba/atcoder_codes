# 118

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc336/tasks/abc336_e)

## 何に着目するか

桁和で割り切れる数は、桁和を固定すれば通常の桁DPで数えられます。

## 解法方針

目標桁和sを全て試し、桁位置・現在和・sでの剰余・上限一致を状態に桁DPします。最終的に和sかつ余り0を加算します。

## 実装要点

先頭0を含めても桁和0のケースを除けば正しく数えられます。

## 計算量

桁数をLとしておおむね `O(L^4)` です。

## 典型・関連問題

典型は **目標値を固定して回す桁DP** です。関連: [ABC235 F](https://atcoder.jp/contests/abc235/tasks/abc235_f)
