# 013

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc252/tasks/abc252_e)

## 何に着目するか

全頂点への最短距離だけでなく、その距離を更新した辺も求めます。

## 解法方針

始点からダイクストラ法を実行します。頂点 `v` の距離が辺 `e` によって更新されたとき、`e` の番号を親辺として記録します。

## 実装要点

優先度付きキューから取り出した距離が最新でない場合は捨てます。等しい距離で更新し直す必要はありません。

## 計算量

`O((N + M) log N)` です。

## 典型・関連問題

典型は **親辺を復元するダイクストラ法** です。関連: [ABC340 D](https://atcoder.jp/contests/abc340/tasks/abc340_d)
