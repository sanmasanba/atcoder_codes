# 043

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc300/tasks/abc300_f)

## 何に着目するか

周期文字列上の条件付き最長区間は、一周期を越える回数と余りに分けられます。

## 解法方針

文字列を二倍にして、各開始位置から許される長さを尺取り法で求めます。周期数を足してクエリ上限で切ります。

## 実装要点

左端ごとに右端を戻さない二ポインタにします。開始位置は一周期内だけ調べます。

## 計算量

`O(N)` です。

## 典型・関連問題

典型は **周期列に対する尺取り法** です。関連: [ABC258 E](018.md)
