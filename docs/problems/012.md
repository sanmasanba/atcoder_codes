# 012

## 問題リンク

[ABC293 E - Geometric Progression](https://atcoder.jp/contests/abc293/tasks/abc293_e)

## 何に着目するか

等比数列の項数が大きく、法は素数とは限りません。逆元による等比数列の公式には依存できません。

## 解法方針

`A^n` と `1 + A + ... + A^(n-1)` を同時に二分累乗で求めます。偶数長は前半の結果を二つ並べ、奇数長は最後の項を追加します。

## 実装要点

再帰・反復のどちらでもよいですが、和とべき乗を組にして返します。すべての演算で法を取ります。

## 計算量

`O(log X)` 時間、再帰なら `O(log X)` スタックです。

## 典型・関連問題

典型は **等比和を含む二分累乗** です。関連: [ABC293 F](https://atcoder.jp/contests/abc293/tasks/abc293_f)
