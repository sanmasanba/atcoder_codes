# 037

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc230/tasks/abc230_e)

## 何に着目するか

`floor(N / i)` は値が変わらない区間を持ち、取り得る値の種類は少数です。

## 解法方針

商が同じ最大の右端を `N // (N // l)` で求め、その区間長と商の積を一度に加算します。

## 実装要点

左端を次の右端の一つ後へ進めます。答えは64bit整数で扱います。

## 計算量

`O(sqrt(N))` です。

## 典型・関連問題

典型は **商が一定の区間への分割** です。関連: [ABC172 D](https://atcoder.jp/contests/abc172/tasks/abc172_d)
