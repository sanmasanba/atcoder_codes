# 111

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc329/tasks/abc329_e)

## 何に着目するか

消去後のワイルドカードは将来の一致を妨げないため、可能な位置をキューで連鎖処理できます。

## 解法方針

各開始位置で文字列Tと矛盾しないかを判定し、可能ならその区間をワイルドカードにします。影響する近傍開始位置を再判定します。

## 実装要点

同じ位置を何度も処理しないよう使用済みを持ちます。最後に全体が消えたか確認します。

## 計算量

`O(NM)` 程度です。

## 典型・関連問題

典型は **局所更新をキューで伝播する文字列消去** です。関連: [ABC315 D](098.md)
