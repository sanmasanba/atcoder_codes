# 104

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc322/tasks/abc322_e)

## 何に着目するか

複数能力の下限を満たす最小コストは、各能力を上限で丸めた多次元DPにできます。

## 解法方針

能力ベクトルをPで打ち切った状態を基数P+1で符号化し、各教材を使う・使わないの最小コストを更新します。

## 実装要点

各次元を独立に `min(P, current + add)` します。無限大の初期値を十分大きく取ります。

## 計算量

`O(N(P+1)^K)` です。

## 典型・関連問題

典型は **上限で丸める多次元ナップサックDP** です。関連: [ABC317 D](099.md)
