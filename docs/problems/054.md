# 054

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc277/tasks/abc277_e)

## 何に着目するか

スイッチ状態により通れる辺が変わるため、頂点だけでは状態を表し切れません。

## 解法方針

各頂点を二つのスイッチ状態へ複製します。通常辺と状態変更辺からなるグラフで0-1 BFSを実行します。

## 実装要点

重み0の辺はdequeの先頭、重み1の辺は末尾に入れます。

## 計算量

`O(N + M)` です。

## 典型・関連問題

典型は **状態を頂点へ持ち上げる0-1 BFS** です。関連: [ABC176 D](https://atcoder.jp/contests/abc176/tasks/abc176_d)
