# 099

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc317/tasks/abc317_d)

## 何に着目するか

勝利へ変える各選挙区は、必要コストと得られる議席数を持つ0/1品物です。

## 解法方針

過半数までに必要な追加議席を求め、議席数を重さとする最小コストナップサックDPを行います。

## 実装要点

既に勝っている選挙区は初期議席へ加えます。必要な説得数は切り上げを正しく計算します。

## 計算量

`O(N sum(Z))` です。

## 典型・関連問題

典型は **価値を満たす最小コストDP** です。関連: [ABC219 D](https://atcoder.jp/contests/abc219/tasks/abc219_d)
