# 003

## 問題リンク

[ABC249 D - Index Trio](https://atcoder.jp/contests/abc249/tasks/abc249_d)

## 何に着目するか

三つ組をそのまま全探索すると間に合いません。値の出現回数を使って、添字ではなく値の組を数えられるか考えます。

## 解法方針

各値の頻度を数えます。`a × b = c` を満たす値の組について `count[a] * count[b] * count[c]` を加算します。

## 実装要点

値域までの頻度配列を作り、存在する `a` と `b` を走査します。積が値域を超えたら打ち切ります。

## 計算量

値域を `M` とすると、おおむね `O(M log M)` 程度です。

## 典型・関連問題

典型は **頻度配列による積の条件の数え上げ** です。関連: [ABC356 E](https://atcoder.jp/contests/abc356/tasks/abc356_e)
