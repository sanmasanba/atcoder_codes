# 059

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc285/tasks/abc285_f)

## 何に着目するか

文字更新後の区間判定では、隣接逆転と文字種の出現範囲を別々に管理します。

## 解法方針

隣接文字の逆転位置をFenwick Treeで数え、各文字の出現位置をFenwick Treeまたはsorted setで管理します。条件を全て満たすか確認します。

## 実装要点

一点更新では前後二か所の隣接関係だけを更新します。

## 計算量

各クエリ `O(26 log N)` です。

## 典型・関連問題

典型は **文字列更新を局所条件と出現範囲へ分解する** 手法です。関連: [ABC157 E](https://atcoder.jp/contests/abc157/tasks/abc157_e)
