# 068

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc229/tasks/abc229_e)

## 何に着目するか

頂点削除後の連結成分数は、頂点追加を逆順に処理すると簡単に更新できます。

## 解法方針

頂点を大きい番号から追加し、既に有効な隣接頂点とUnion-Findで併合します。成分数を記録して逆順に出力します。

## 実装要点

頂点を追加した時点で成分数を一つ増やし、異なる成分を併合するたび一つ減らします。

## 計算量

`O((N+M)α(N))` です。

## 典型・関連問題

典型は **削除を逆順追加にするオフラインUnion-Find** です。関連: [ABC264 E](021.md)
