# 041

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc288/tasks/abc288_d)

## 何に着目するか

長さKの操作は、添字をKで割った余りごとに独立した差分へ影響します。

## 解法方針

差分配列を作り、各剰余類ごとの累積和を前処理します。クエリ区間の内部差分が全て0にできるかを剰余ごとに確認します。

## 実装要点

端点に対応する差分だけは別扱いです。区間を半開区間として一貫して実装します。

## 計算量

前処理 `O(N)`、各クエリ `O(K)` です。

## 典型・関連問題

典型は **差分配列を剰余類で分解する** 手法です。関連: [ABC133 D](https://atcoder.jp/contests/abc133/tasks/abc133_d)
