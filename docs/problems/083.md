# 083

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc279/tasks/abc279_f)

## 何に着目するか

箱の併合・分離とボール番号の問い合わせでは、実体とラベルを分離します。

## 解法方針

各集合の根をUnion-Find風の親で持ち、箱が指す根とボールが指すノードを別配列で管理します。併合では根の向きを付け替えます。

## 実装要点

新しいボールには新ノードを発行します。箱が空になると箱側の参照を空にします。

## 計算量

各操作はならし `O(α(Q))` です。

## 典型・関連問題

典型は **ラベルと集合実体を分けるUnion-Find** です。関連: [ABC228 D](028.md)
