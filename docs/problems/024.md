# 024

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc255/tasks/abc255_e)

## 何に着目するか

隣り合う和から数列を復元すると、最初の値以外は一つの変数で表せます。

## 解法方針

`A[i]` を `x` の符号付き一次式として表します。各目標値がどの `x` のときに現れるかを頻度表に加え、頻度最大の `x` が答えです。

## 実装要点

係数の符号は位置の偶奇で交互に変わります。`x` の候補は整数なので辞書で数えられます。

## 計算量

`O(NM)` 時間、`O(NM)` 空間です。

## 典型・関連問題

典型は **漸化式を一変数に落とす頻度数え上げ** です。関連: [ABC165 C](https://atcoder.jp/contests/abc165/tasks/abc165_c)
