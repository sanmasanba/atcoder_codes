# 075

## 問題リンク

[ABC248 E - K-colinear Line](https://atcoder.jp/contests/abc248/tasks/abc248_e)

## キーワード

同じ幾何図形を重複なく数えるなら、整数係数で一意に正規化する

## 何に着目するか

`K≥2` なら、条件を満たす直線は必ず与えられた二点を通ります。したがって全点対から候補直線を作れば漏れません。同じ直線を異なる点対から何度も数えないため、表現を一意にします。

## 解法方針

二点 `(x1,y1),(x2,y2)` を通る直線を

```text
a*x + b*y + c = 0
```

の整数係数で表します。方向差を `dx=x2-x1`, `dy=y2-y1` とし、`g=gcd(|dx|,|dy|)` で割って

```text
a = dy/g
b = -dx/g
c = -(a*x1 + b*y1)
```

とします。最後に、`a<0` または `a==0 and b<0` なら三係数を全て `-1` 倍して符号を統一します。

```text
同じ直線の別の二点対
  → 同じ (a,b,c) へ正規化
  → set で一つにまとまる
```

全点対から得た直線 ID を `set` に入れ、各直線について全点が式を満たすかを数えます。点数が `K` 以上なら答えへ加えます。

`K=1` では一点を通る直線が無限本あるので、特別に `Infinity` を出力します。

## tips

### 実装

点の座標が整数なので、傾きや切片の浮動小数点比較は不要です。係数 `(a,b,c)` をタプルにして `set` へ入れます。

二点が重複しない制約を利用し、`g=0` のケースはありません。

### よくある誤り

- 傾きを分数／浮動小数点で持つ。同じ直線の比較で誤差や垂直線の例外が出ます。
- 方向ベクトルの符号を統一しない。逆向きの二点対で別 ID になります。
- `K=1` を通常の数え上げへ入れる。答えは有限個ではありません。

### 計算量

候補直線は `O(N^2)` 本、各直線で `N` 点を確認します。時間 `O(N^3)`、メモリ `O(N^2)` です。
