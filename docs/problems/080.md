# 080

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc272/tasks/abc272_e)

## 何に着目するか

各要素が特定時刻に取り得る値だけを、その時刻のバケットへ配れます。

## 解法方針

要素ごとに値が0以上N以下となる時刻範囲を求め、その範囲の時刻バケットへ値を追加します。各バケットでmexを求めます。

## 実装要点

各要素の追加回数は `O(N / i)` 型に抑えられます。バケット内はsetまたは印配列でmexを取ります。

## 計算量

全体でおおむね `O(N log N)` です。

## 典型・関連問題

典型は **有効な時刻だけへ配るオフラインバケット** です。関連: [ABC330 E](https://atcoder.jp/contests/abc330/tasks/abc330_e)
