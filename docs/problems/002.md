# 002

## 問題リンク

[ABC231 C - Counting 2](https://atcoder.jp/contests/abc231/tasks/abc231_c)

## 何に着目するか

各クエリで「ある値以上の要素数」を数える処理を、毎回線形探索していないか確認します。

## 解法方針

配列を一度ソートし、各クエリ値以上となる最初の位置を二分探索します。その位置より右にある要素が答えです。

## 実装要点

Python では `bisect_left(A, x)` を使い、答えを `N - index` とします。

## 計算量

前処理は `O(N log N)`、各クエリは `O(log N)` です。

## 典型・関連問題

典型は **ソート済み配列への二分探索** です。関連: [ABC330 E](https://atcoder.jp/contests/abc330/tasks/abc330_e)
