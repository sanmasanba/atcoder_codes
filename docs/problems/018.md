# 018

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc258/tasks/abc258_e)

## 何に着目するか

同じ剰余位置から始めれば、次の位置と選ぶ個数は常に同じです。大きなクエリ番号を直接シミュレーションしません。

## 解法方針

二倍にした配列への尺取り法で、各開始位置の個数と次の開始位置を前計算します。開始位置の遷移に対してダブリングを作り、各クエリの位置へ飛びます。

## 実装要点

合計が一周分を超える部分は、全周回数を先に加算してから残りを尺取りします。添字は常に `N` で剰余を取ります。

## 計算量

前処理 `O(N log N)`、各クエリ `O(log N)` です。

## 典型・関連問題

典型は **尺取り法で作る状態遷移＋ダブリング** です。関連: [ABC241 E](009.md)
