# 082

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc274/tasks/abc274_e)

## 何に着目するか

訪問済み集合と現在位置で状態を持ち、加速器の数が移動速度を決めます。

## 解法方針

bit DPで各地点への最短実時間を更新します。次の移動距離を、現在までに取った加速器数に応じた速度で割ります。

## 実装要点

目標地点をすべて訪れた状態のうち、現在位置を問わない最小を取ります。

## 計算量

`O(2^N N^2)` 程度です。

## 典型・関連問題

典型は **付加状態を含む巡回bit DP** です。関連: [ABC190 E](https://atcoder.jp/contests/abc190/tasks/abc190_e)
