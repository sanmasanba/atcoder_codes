# 016

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc256/tasks/abc256_e)

## 何に着目するか

各頂点から出る辺が一つだけのグラフでは、閉路ごとに必要な選択が一つにまとまります。

## 解法方針

未処理の頂点から辺をたどり、同じ探索中に再訪した頂点があれば閉路を見つけます。その閉路に含まれるコストの最小値を答えへ加えます。

## 実装要点

探索開始ごとに訪問順を記録すると、閉路に入った位置を特定できます。すでに完了した頂点からは始めません。

## 計算量

`O(N)` 時間・空間です。

## 典型・関連問題

典型は **functional graph の閉路抽出** です。関連: [ABC357 E](https://atcoder.jp/contests/abc357/tasks/abc357_e)
