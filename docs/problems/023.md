# 023

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc267/tasks/abc267_e)

## 何に着目するか

頂点を消す順序のコストの最大値を最小化します。ある上限以下で全頂点を消せるかは単調です。

## 解法方針

上限を固定し、隣接頂点の重み和が上限以下の頂点をキューに入れて削除します。全頂点を削除できるかを判定して、答えを二分探索します。

## 実装要点

頂点を削除したら隣接頂点の和を減らし、新たに条件を満たした頂点をキューへ入れます。各頂点は一度だけ削除します。

## 計算量

判定は `O(N + M)`、全体は `O((N + M) log S)` です。

## 典型・関連問題

典型は **単調判定を持つ頂点削除貪欲** です。関連: [ABC239 E](https://atcoder.jp/contests/abc239/tasks/abc239_e)
