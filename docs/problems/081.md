# 081

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc273/tasks/abc273_e)

## 何に着目するか

以前の状態へ戻る操作があるため、状態を破壊せず親ポインタとして残します。

## 解法方針

追加操作ごとに新しいノードを作り、前のノードを親にします。削除は親へ戻り、保存・読込はノードIDを辞書に記録・復元します。

## 実装要点

空状態を番兵ノードにし、各ノードに現在値を持たせます。

## 計算量

各操作は期待 `O(1)` です。

## 典型・関連問題

典型は **親ポインタで作る永続スタック** です。関連: [ABC315 E](https://atcoder.jp/contests/abc315/tasks/abc315_e)
