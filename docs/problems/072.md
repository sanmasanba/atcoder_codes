# 072

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc238/tasks/abc238_e)

## 何に着目するか

区間和が指定値になる条件は、prefix和の二点が同じ値になる関係です。

## 解法方針

prefix位置を頂点とし、各区間条件を二頂点間の辺にします。0からNへ到達できるかをDFSまたはUnion-Findで判定します。

## 実装要点

区間 `[l, r]` はprefix位置 `l-1` と `r` を結びます。添字のずれに注意します。

## 計算量

`O(N+Q)` です。

## 典型・関連問題

典型は **prefix和条件をグラフの連結性へ変換する** 手法です。関連: [ABC087 D](https://atcoder.jp/contests/abc087/tasks/arc090_b)
