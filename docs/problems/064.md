# 064

## 何に着目するか

ソート指示前は挿入順、指示後は最小値順という二種類の順序を分けます。

## 解法方針

通常キューと最小ヒープを持ちます。ソート指示でキュー内の全要素をヒープへ移し、出力時はヒープを優先します。

## 実装要点

キューにはdequeを使い、ヒープへの移動は各要素一度だけにします。

## 計算量

全体で `O(Q log Q)` です。

## 典型・関連問題

典型は **遅延して順序を切り替える二容器管理** です。関連: [ABC212 D](001.md)
