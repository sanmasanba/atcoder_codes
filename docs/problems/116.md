# 116

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc334/tasks/abc334_e)

## 何に着目するか

一つのマスを追加した後の連結成分数は、その隣接成分の種類数だけで決まります。

## 解法方針

既存の赤マスをUnion-Findで成分化します。各白マスについて隣接する異なる根数を数え、成分数の変化を平均します。

## 実装要点

隣接根は小さなsetへ入れて重複を除きます。期待値は白マス数の逆元を掛けます。

## 計算量

`O(HW α(HW))` です。

## 典型・関連問題

典型は **局所的な成分変化の期待値** です。関連: [ABC305 E](094.md)
