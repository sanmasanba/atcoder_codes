# 049

## 問題リンク

[ABC218 D - Rectangles](https://atcoder.jp/contests/abc218/tasks/abc218_d)

## キーワード

格子点の長方形は、対角の二点から残りの頂点を確認する

## 何に着目するか

軸に平行な長方形は、左下と右上の対角二点を選べば、残りの二点が一意に決まります。点集合にその二点が含まれるかを高速に判定できれば、候補の対角だけを全探索できます。

## 解法方針

全点を `(x,y)` の組として `set` へ入れます。二点 `(x1,y1)`, `(x2,y2)` を選び、`x1<x2` かつ `y1<y2` のときだけ、この二点を左下・右上とみなします。

必要な残り二点は `(x1,y2)` と `(x2,y1)` です。

```text
(x1,y2) ●────● (x2,y2)
        │    │
(x1,y1) ●────● (x2,y1)
```

二点とも集合にあれば長方形が一つあります。対角の向きを `x1<x2, y1<y2` に固定したため、一つの長方形をちょうど一回だけ数えます。

## tips

### 実装

二重ループで `i<j` の点対を調べ、条件を満たすときだけ `in` で二点を確認します。Python のタプルを `set` に入れれば期待 `O(1)` です。

`x` または `y` が同じ二点は対角にできないので、条件で除きます。

### よくある誤り

- 全ての点対を対角とみなし、同じ長方形を二回数える。対角の上下方向を固定します。
- 片方の残り頂点だけを確認する。四隅全てが必要です。
- `x1>x2` や `y1>y2` の場合に座標を正規化せず、そのまま残り頂点を作る。

### 計算量

点対を `O(N^2)` 個調べ、各集合検索は期待 `O(1)` です。時間 `O(N^2)`、メモリ `O(N)` です。
