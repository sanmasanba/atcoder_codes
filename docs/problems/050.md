# 050

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc246/tasks/abc246_d)

## 何に着目するか

二変数式が両変数について単調増加なら、全探索を二ポインタに落とせます。

## 解法方針

`a` を小さい方から動かし、条件を満たす最小の `b` を大きい方から減らして探します。候補値の最小を更新します。

## 実装要点

`b` は次の `a` で増やす必要がありません。式と答えは64bit整数で計算します。

## 計算量

`O(M)` です。

## 典型・関連問題

典型は **単調な二変数関数への二ポインタ** です。関連: [ABC296 D](https://atcoder.jp/contests/abc296/tasks/abc296_d)
