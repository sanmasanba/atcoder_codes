# 021

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc264/tasks/abc264_e)

## 何に着目するか

辺削除を順方向で処理すると連結成分を分割する必要があります。操作を逆にすると辺追加になります。

## 解法方針

削除されない辺だけでUnion-Findを作り、削除クエリを逆順に戻します。発電所を一つの特別頂点にまとめ、その成分に属する都市数の増減を記録します。

## 実装要点

二つの成分を併合する直前に、それぞれが特別頂点を含むか確認します。二重に削除される辺にも注意します。

## 計算量

`O((N + M + Q) α(N))` です。

## 典型・関連問題

典型は **削除クエリの逆順Union-Find** です。関連: [ABC120 D](https://atcoder.jp/contests/abc120/tasks/abc120_d)
