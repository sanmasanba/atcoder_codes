# 019

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc262/tasks/abc262_d)

## 何に着目するか

選ぶ個数ごとに、和をその個数で割った余りだけが必要です。

## 解法方針

選ぶ個数を `k` と固定し、`dp[j][r]` を `j` 個選んで和を `k` で割った余りが `r` となる通り数にします。全要素を一度ずつ使う0/1 DPです。

## 実装要点

同じ要素を複数回選ばないよう、選択数 `j` は大きい方から更新します。各 `k` の答えは `dp[k][0]` です。

## 計算量

`O(N^4)` 時間、`O(N^2)` 空間です。

## 典型・関連問題

典型は **選択数ごとに法を変える剰余DP** です。関連: [ABC281 D](https://atcoder.jp/contests/abc281/tasks/abc281_d)
