# 058

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc282/tasks/abc282_e)

## 何に着目するか

全頂点対に重みがある完全グラフでも、最大全域木はKruskal法で求められます。

## 解法方針

全ての辺重みを計算して降順にソートし、Union-Findで閉路を作らない辺を採用します。

## 実装要点

べき乗は組込みの剰余付き累乗を使い、二方向の値の和を法で取ります。

## 計算量

`O(N^2 log N)` です。

## 典型・関連問題

典型は **最大全域木（降順Kruskal）** です。関連: [ABC218 E](https://atcoder.jp/contests/abc218/tasks/abc218_e)
