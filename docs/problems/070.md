# 070

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc234/tasks/abc234_f)

## 何に着目するか

各文字を何個使うかを選べば、並べ方は組合せで増やせます。

## 解法方針

文字種を順に処理し、現在長に新しい文字をj個挿入する組合せ数を掛けたDPを行います。

## 実装要点

階乗・逆階乗を前計算し、空文字列を最後に除きます。

## 計算量

`O(26N^2)` です。

## 典型・関連問題

典型は **文字頻度ごとの組合せDP** です。関連: [ABC171 F](https://atcoder.jp/contests/abc171/tasks/abc171_f)
