# 061

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc201/tasks/abc201_d)

## 何に着目するか

両者が最善を尽くす交互ゲームでは、各マスでの得失点差を後ろから決められます。

## 解法方針

ゴールから逆向きに、先手番なら次の差を最大化、後手番なら最小化します。移動先のマスの点を加減します。

## 実装要点

右と下がないゴールの値を0にし、盤面を右下から走査します。

## 計算量

`O(HW)` です。

## 典型・関連問題

典型は **minimax の得失点差DP** です。関連: [ABC270 D](https://atcoder.jp/contests/abc270/tasks/abc270_d)
