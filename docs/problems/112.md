# 112

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc330/tasks/abc330_f)

## 何に着目するか

移動量D以内で一辺Lの正方形へ入れられるかはLに関して単調です。

## 解法方針

Lを二分探索します。x座標・y座標を独立にソートし、長さLの区間へ集める最小移動量を中央値とprefix和で求めます。

## 実装要点

区間左端候補は座標から導けます。二軸の必要量を足してD以下か判定します。

## 計算量

`O(N log N log C)` 程度です。

## 典型・関連問題

典型は **中央値・prefix和を使う答え二分探索** です。関連: [ABC257 D](017.md)
