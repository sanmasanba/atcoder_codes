# 093

## 問題リンク

[問題を開く](https://atcoder.jp/contests/abc304/tasks/abc304_e)

## 何に着目するか

禁止条件は個別頂点でなく、初期連結成分の組に対して課されています。

## 解法方針

既存辺でUnion-Findを作り、禁止頂点対を根IDの組としてsetへ保存します。クエリ辺の両端の根組が禁止かを確認します。

## 実装要点

根IDの組は順序を揃えて保存します。既に同一成分なら許可です。

## 計算量

前処理・各クエリとも期待 `O(α(N))` です。

## 典型・関連問題

典型は **縮約成分対への禁止条件** です。関連: [ABC264 E](021.md)
